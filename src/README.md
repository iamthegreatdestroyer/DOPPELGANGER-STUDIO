# Source Code

This directory contains all the source code for DOPPELGANGER STUDIO.

## Structure

- **`core/`** - Core application logic and base classes
- **`services/`** - Microservices for different functionality
  - `research/` - Show research and data acquisition
  - `creative/` - AI-powered content generation
  - `animation/` - Video animation and rendering
  - `voice/` - Voice synthesis
  - `asset_manager/` - Asset scraping and management
  - `publisher/` - Content publishing and distribution
- **`ai_engine/`** - AI/ML model integrations
- **`ui/`** - User interface (PyQt6)
- **`api/`** - REST API endpoints
- **`utils/`** - Utility functions and helpers
- **`models/`** - Data models and schemas

## Development

All code should follow the standards defined in `docs/INSTRUCTIONS.md`.

## Testing

Tests are located in the `tests/` directory at the root level.
